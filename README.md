# Azure Secure Infrastructure Project

## Overview
This project demonstrates a secure Azure environment using private networking and identity-based access.

## Key Components

### Compute
- Virtual Machine (VM-Simone)

### Networking
- Virtual Network
- Network Security Group (NSG)
- Azure Bastion
- Private Endpoints
- Private DNS Zones

### Security
- Azure Key Vault
- Managed Identity

### Storage
- Azure Storage Account (private access)
